# ORYA Jewelry Store 💎

ORYA هو موقع إلكتروني فاخر لبيع المجوهرات، يحتوي على لوحة تحكم كاملة للمشرفين (Admin Dashboard)، ومتجر متكامل للمستخدمين لشراء المنتجات، بالإضافة إلى مساعد ذكي (Chatbot) يعتمد على الذكاء الاصطناعي (RAG Workflow) لمساعدة الزوار.

---

## 🚀 خطوات التشغيل على جهاز جديد

لتشغيل هذا المشروع على جهازك، يرجى اتباع الخطوات التالية بالترتيب:

### 1. تثبيت المتطلبات
تأكد من تثبيت Python على جهازك، ثم قم بتثبيت المكتبات المطلوبة عبر الأمر التالي:
```bash
pip install -r requirements.txt
```

### 2. إعداد ملف البيئة (Environment Variables)
1. قم بإنشاء ملف جديد في المجلد الرئيسي وتسميته `.env`.
2. انسخ محتويات الملف التوضيحي `.env.example` وضعها داخل الملف الجديد `.env`.
3. قم بتعبئة البيانات المطلوبة (مثل مفتاح OpenAI API لتشغيل الشات بوت).

### 3. إنشاء وتغذية قاعدة البيانات (Database Seeding)
لتوليد قاعدة بيانات جديدة وإدخال المنتجات الافتراضية وحسابات التجربة، قم بتشغيل السكربتات التالية:
```bash
# إنشاء الحسابات الأساسية والمنتجات التجريبية
python seed_db.py

# إضافة كوبونات الخصم، الطلبات، والمراجعات الإضافية للتجربة
python seed_data.py
```

### 4. تشغيل الموقع
الآن يمكنك تشغيل سيرفر التطوير المحلي عبر الأمر التالي:
```bash
python run.py
```
ثم افتح المتصفح وتوجه إلى الرابط: `http://127.0.0.1:5000`

*   **اسم المستخدم (Username):** `admin`
*   **البريد الإلكتروني:** `admin@orya.com`
*   **كلمة المرور:** `1234`
